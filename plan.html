Mission Statement

This website has been created in order to increase the business of Diazepam Cafe. The old website wasn't clear enough, so we made a lot of improvement so that clients are able to find the products very easily.

Success Evaluation

In order to measure the success of the website, we will count the number of orders before the new website and after, and see if it increased.

Content

Home Page: this page will display the best sellers of the cafe, the sales available and a picture of the new outlet.

Our menu: this page will display the list of each product available to order from Diazepam Cafe:

Ice-cream scoops: this page will display the different flavors available, a picture of the product, and details about the product and how to order it.

Target Audience

The group age aimed by the website is from 13 years old to 70+. The design of the page will make the navigation very easy for younger and older users, so that everyone can use the website without having troubles to find the products. The colors are light in order to attract young audience.
